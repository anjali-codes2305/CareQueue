# CareQueue

Intelligent Hospital Queue & Patient Flow Platform. Replace static token systems with adaptive, data-driven hospital workflows. Real-time queue management with role-based access for every team member.

## Features

- **Patient Portal**: Track queue position in real-time.
- **Nurse Triage**: Approve or reject priority cases with severity scoring.
- **Doctor Console**: One-click patient intake and consultation metrics.
- **Reception Desk**: Patient onboarding, records management, and automated WhatsApp updates.
- **Admin Analytics**: High-level load charts and performance tracking.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Configure Supabase credentials in your `.env` file to enable authentication and database modules.
